# 📊 차트 시각화 예시 - 사용자 화면

이 문서는 헬스장 회원관리 시스템에서 사용자가 보게 될 차트와 그래프의 시각적 예시를 보여줍니다.

---

## 1. 신체스펙 변화 추이 차트 (라인 차트)

### 화면 구성

```
┌─────────────────────────────────────────────────────────────────┐
│  회원 상세 > 능력치 분석                                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  신체스펙 변화 추이                                             │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │                                                         │  │
│  │  80 ┤                                                  │  │
│  │     │                                                  │  │
│  │  75 ┤     ●──────●                                     │  │
│  │     │    ╱        ╲                                     │  │
│  │  70 ┤   ●          ●──────●                             │  │
│  │     │  ╱                  ╲                             │  │
│  │  65 ┤ ●                    ●──────●                     │  │
│  │     │                            ╲                     │  │
│  │  60 ┤                              ●                    │  │
│  │     │                              │                    │  │
│  │  55 ┤                              │                    │  │
│  │     └───────────────────────────────────────────────    │  │
│  │      1월   2월   3월   4월   5월   6월                  │  │
│  │                                                         │  │
│  │  범례:                                                  │  │
│  │  ──── 몸무게 (kg)                                       │  │
│  │  ──── 체지방률 (%)                                      │  │
│  │  ──── 근육량 (kg)                                       │  │
│  └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│  [마우스를 차트 위에 올리면 상세 정보 표시]                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 실제 데이터 예시

**백엔드 API 응답**:
```json
{
  "bodySpecs": [
    { "date": "2024-01-01", "weight": 75, "bodyFat": 15, "muscleMass": 60 },
    { "date": "2024-02-01", "weight": 74, "bodyFat": 14, "muscleMass": 61 },
    { "date": "2024-03-01", "weight": 73, "bodyFat": 13, "muscleMass": 62 },
    { "date": "2024-04-01", "weight": 72, "bodyFat": 12, "muscleMass": 63 },
    { "date": "2024-05-01", "weight": 71, "bodyFat": 11, "muscleMass": 64 },
    { "date": "2024-06-01", "weight": 70, "bodyFat": 10, "muscleMass": 65 }
  ]
}
```

**사용자가 보는 화면**:
- X축: 측정 날짜 (1월, 2월, 3월...)
- Y축: 수치 (몸무게, 체지방률, 근육량)
- 3개의 라인: 각각 다른 색상으로 표시
- 호버 시: 해당 날짜의 정확한 수치 표시

## 주요 특징

### 1. 인터랙티브 요소
- **호버 효과**: 마우스를 차트 위에 올리면 상세 정보 표시
- **클릭 가능**: 특정 데이터 포인트 클릭 시 상세 정보 모달
- **줌/팬**: 차트 확대/축소 기능 (선택사항)

### 2. 반응형 디자인
- **데스크톱**: 큰 화면에 여러 차트를 한 번에 표시
- **태블릿**: 2열 레이아웃
- **모바일**: 1열 레이아웃, 스크롤 가능

### 3. 색상 구분
- **신체스펙**: 파란색 계열
- **근력 테스트**: 보라색 계열
- **심폐지구**: 초록색 계열
- **평균 비교**: 주황색 계열

### 4. 데이터 표시
- **툴팁**: 호버 시 정확한 수치 표시
- **범례**: 각 데이터 시리즈 설명
- **축 레이블**: 명확한 단위 표시 (kg, %, bpm 등)

## 기술 구현

이 모든 차트는 **Recharts** 라이브러리를 사용하여 구현됩니다.

- **LineChart**: 신체스펙, 심폐지구 테스트
- **BarChart**: 근력 테스트, 평균 비교
- **ComposedChart**: 평균 비교 (바 + 라인 조합)

자세한 구현 방법은 `docs/CHART_GUIDE.md`를 참고하세요.
