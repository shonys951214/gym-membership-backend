## 핵심 메시지

### 평가의 본질

**평가 종목은 '정확한 수치'보다 회원에게 운동을 시켜도 되는지를 판단하는 도구**

- ❌ "누가 제일 세냐?"
- ✅ "이 사람이 지금 뭘 해도 되는 상태인가?"
- ✅ "어떻게 움직이고, 무엇을 시켜야 하는가?"

**현장은 안전 → 기능 → 체력 → 결과 순으로 테스트를 고른다**

### 헬스인들이 평가 종목을 고르는 사고 구조

생활체육지도사·PT 트레이너는 이렇게 생각함:

1. **통증·위험 있나?** → 있으면 제한
2. **기본 움직임 되나?** → 안 되면 교정
3. **얼마나 오래 할 수 있나?** → 지구력
4. **어느 정도 힘을 쓸 수 있나?** → 근력
5. **그럼 몸은 바뀌고 있나?** → 체성분

➡️ 그래서 평가 종목도 이 순서로 배치됨

### 2단계 평가 레이어 구조

**초기 평가 레이더 차트는 '요약 지표'로 고정하고, 정기 평가의 상세 데이터들을 '초기 평가 항목으로 환산'해 누적 반영한다**

- 초기 평가 = 공통 언어 (6개 축 고정)
- 정기 평가 = 데이터 원천 (상세 지표 확장 가능)
- 초기 평가도 1회차 정기 평가로 취급

## 평가 체계 개요

### 평가의 목적 및 철학

- 능력 비교가 아닌 운동 처방 기준점 설정
- **평가 종목은 '정확한 수치'보다 회원에게 운동을 시켜도 되는지를 판단하는 도구**
- 부상 위험 관리 최우선
- 개인 맞춤 운동 프로그램 설계 기반

### 2단계 평가 레이어 구조

#### Layer 1: 상위 평가 축 (고정) - 레이더 차트용

**6대 영역 (절대 변경 불가)**:
1. 하체 근력 (STRENGTH)
2. 심폐 지구력 (CARDIO)
3. 근지구력 (ENDURANCE)
4. 유연성 (FLEXIBILITY)
5. 체성분 밸런스 (BODY)
6. 부상 안정성 (STABILITY)

**특징**:
- 레이더 차트 전용
- 서비스의 "공용 언어"
- 초기 평가에서 정의, 이후 변경 없음
- 모든 평가에서 동일한 기준으로 계산

#### Layer 2: 하위 상세 지표 (가변) - 그래프 차트용

**정기 평가에서 늘어나는 항목들**:
- 하체 근력: 레그프레스 10RM, 스쿼트 5RM, 데드리프트 3RM, 점프 파워, 좌/우 밸런스
- 심폐 지구력: VO2max, 러닝머신 테스트, 자전거 테스트
- 근지구력: 플랭크 시간, 푸쉬업 횟수, 버피 테스트
- 유연성: 좌전굴, 어깨 가동, 고관절 가동
- 체성분: 인바디 세부 지표
- 안정성: OHSA, Y-Balance, 한 발 서기

**특징**:
- 그래프 차트용
- 비교 분석용
- AI 분석 재료
- 확장 가능

### 평가 위계 구조 (실제 사용 흐름)

#### ① 정적 평가 (Baseline / 위험 필터링)

- 목적: "이 사람을 움직여도 되는가?"
- 항목:
  - 설문 (PAR-Q, 통증, 병력)
  - 체성분 (인바디)
  - 체형 정렬 (정면/측면/후면)
- 역할: 위험 필터링 및 정보 수집 (점수 계산과 무관)

#### ② 동적 평가 (실제 운동 처방의 핵심)

- 목적: "어떻게 움직이고, 무엇을 시켜야 하는가?"
- 항목:
  - 유연성
  - 근력
  - 밸런스
  - 심폐능력
- 점수화: 레이더 차트의 주 점수

### 평가 주기

- 초기 평가: 신규 회원 등록 시 필수 (모든 항목 평가, 기준선 설정)
- 정기 평가:
  - 1개월 후: 빠른 변화 확인 (선택적)
  - 3개월 후: 주요 개선 효과 확인 (권장) - SBD 1RM 추가 가능
  - 6개월 후: 장기적 변화 추적 (필수)

### 레이더 차트 활용

- 6개 축으로 구성된 레이더 차트
- 초기 평가(회색) vs 현재 평가(색상) 비교
- **회원 설명**: "이건 성적표가 아니라 운동 설계 지도입니다"
- 월별/분기별 추이 시각화
- **점수 표시**: 각 축에 점수(0-100) 숫자로 표시

## 6개 평가 영역별 상세 설명

### 1. 부상 안정성 (STABILITY) - 20% ⭐ 최우선

**현장 인식**: 최우선 (운동 가능 여부)

**🎯 목적**:
- 진짜 목적은 점수 ❌
- 👉 **"이 회원, 오늘 뭐 시켜도 되나?"**

**평가 종목**:
- 설문 (통증, 수술, 불편)
- 정적 자세 관찰 (정면/측면/후면)
- 기본 동작 관찰 (앉기, 팔 들기, 스쿼트 패턴)
- 한 발 서기 테스트 (초) - 선택적
- 눈 감고 균형 잡기 (초) - 선택적

**평가 방법**:
- **수치보다 O / △ / X 개념**
- 관찰 위주, 간단한 테스트
- 정적 평가 결과와 연계하여 운동 제한 플래그 설정

**현장 논리**:
- "여기서 걸리면 근력·심폐 평가 점수는 상한선 걸린다"
- 평가 종목은 간단 + 관찰 위주
- 부상 이력과 연계 평가

**상/중/하 기준**:
- 상 (O): 모든 동작 제한 없음, 통증 없음
- 중 (△): 경미한 불편, 일부 동작 주의 필요
- 하 (X): 통증 있음, 특정 동작 제한 필요

**가중치 이유**: 이거 낮으면 다 의미 없음

---

### 2. 심폐 지구력 (CARDIO) - 20%

**현장 인식**: 건강·다이어트 핵심

**🎯 목적**:
- 건강·다이어트
- 운동 지속 능력

**평가 종목**:
- 스텝 테스트 (하버드 스텝 테스트 - 심박수 회복 속도)
- 자전거 (선택적)
- 걷기 기반 테스트 (선택적)
- 안정 시 심박수
- 회복 심박수 (운동 후 1분, 3분)

**평가 방법**:
- 심박수 측정 및 회복 시간 평가
- 연령별 최대 심박수 대비 회복률

**현장 논리**:
- "심폐 낮으면 근력 운동도 금방 무너진다"
- **최대 테스트 ❌**
- 회복 능력 위주

**상/중/하 기준**:
- 상: 연령 대비 상위 25%, 빠른 회복
- 중: 연령 대비 중간 50%, 보통 회복
- 하: 연령 대비 하위 25%, 느린 회복

**가중치 이유**: 건강, 체중관리 핵심

---

### 3. 근지구력 (ENDURANCE) - 20%

**현장 인식**: 일반인 체력의 핵심, PT 체감 만족도 1위

**🎯 목적**:
- 일반인 체력의 핵심
- 실제 운동 수행 능력

**평가 종목**:
- 푸쉬업 (1분간 횟수 또는 최대 횟수)
- 플랭크 (유지 시간)
- 반복 가능한 머신 동작 (예: 레그 프레스 20회)
- 윗몸 일으키기 (1분간 횟수) - 선택적

**평가 방법**:
- 시간 제한 내 최대 반복 횟수 측정
- 연령/성별 평균 대비

**현장 논리**:
- "근지구력 좋으면 운동을 '잘 버틴다'"
- 장비 적고, 실패해도 안전
- 수업 설계에 바로 쓰임

**상/중/하 기준**:
- 상: 연령/성별 상위 25%
- 중: 연령/성별 중간 50%
- 하: 연령/성별 하위 25%

**가중치 이유**: 일반인 체력의 본질

**참고**: 근지구력 + 심폐지구력 = 40% (이 조합은 거의 고정값)

---

### 4. 하체 근력 (STRENGTH) - 15%

**현장 인식**: 기능적 체력의 중심

**🎯 목적**:
- 기능적 움직임의 중심
- 전신 운동 수행 가능 여부

**평가 종목**:
- 스쿼트 패턴 (동작 관찰 + 반복 횟수) - 초기 평가
- 체어 스탠드 (의자에서 일어서기 횟수) - 초기 평가
- 안전한 중량 반복 (예: 레그 프레스 10RM 추정) - 초기 평가
- **스쿼트 5RM** - 정기 평가 (조건부)
- **벤치프레스 5RM** - 정기 평가 (조건부)
- **데드리프트 5RM** - 정기 평가 (조건부)
- 악력 측정 (kg) - 선택적
- 팔굽혀펴기 (1분간 횟수) - 선택적

**평가 방법**:
- 측정값을 연령/성별 기준표와 비교
- **1RM 직접 측정 ❌**
- 추정 or 반복 기준

**현장 논리**:
- "하체 근력은 최대 힘이 아니라 안정된 반복"
- 초기 평가: 안전한 패턴 관찰
- 정기 평가: SBD 1RM 추가 (조건부)

**상/중/하 기준**:
- 상: 연령/성별 상위 25%, 안정된 반복 가능
- 중: 연령/성별 중간 50%, 기본 반복 가능
- 하: 연령/성별 하위 25%, 반복 어려움

**가중치 이유**: 기능적 움직임의 중심

**주의사항**: 
- 1RM은 위험 (일반인은 반복 수행 능력이 더 중요)
- 초기 평가에서는 SBD 측정하지 않음

---

### 5. 체성분 밸런스 (BODY) - 15%

**현장 인식**: 결과 지표

**🎯 목적**:
- 회원 설득
- 변화 체감

**평가 종목**:
- 인바디 (체지방률, 근육량, BMI)
- 좌우 밸런스 (선택적)
- 허리둘레 (cm) - 선택적

**평가 방법**:
- 인바디 기기 또는 체성분 분석
- 연령/성별 표준 범위 비교

**현장 논리**:
- "이건 평가라기보다 결과 리포트다"
- **점수는 절대값보다 변화량**

**상/중/하 기준**:
- 상: 연령/성별 건강 범위 내, 개선 추이 양호
- 중: 경계선 범위, 개선 추이 보통
- 하: 건강 범위 벗어남, 개선 필요

**가중치 이유**: 회원이 체감하는 결과

---

### 6. 유연성 (FLEXIBILITY) - 10%

**현장 인식**: 보조적이지만 필수

**🎯 목적**:
- 유연성 자체보다
- 👉 **관절 가동 가능 범위 확인**

**평가 종목**:
- 좌전굴 (앉아 윗몸 앞으로 굽히기) (cm)
- 어깨 가동 (손끝 간 거리)
- 고관절 가동 (전각도)
- 발목 가동 (선택적)

**평가 방법**:
- 관절 가동 범위 측정
- 정상 범위 대비 평가

**현장 논리**:
- "유연성 점수가 낮으면 근력 테스트를 못 하게 되는 경우가 많다"
- 유연성은 독립 점수 + 제한 조건 생성용

**상/중/하 기준**:
- 상: 정상 범위 이상
- 중: 정상 범위 내
- 하: 정상 범위 미만 (근력 테스트 제한 가능)

**가중치 이유**: 부상 예방·자세 보조

## 가중치 선정 기준 및 이유

### 수정된 가중치 (현장 기준)

| 평가 항목 | 가중치 | 이유 |
|---------|--------|------|
| 부상 안정성 | 20% | 이거 낮으면 다 의미 없음 (운동 가능 여부) |
| 심폐 지구력 | 20% | 건강, 체중관리 핵심 |
| 근지구력 | 20% | 일반인 체력의 본질 |
| 하체 근력 | 15% | 기능적 움직임의 중심 |
| 체성분 밸런스 | 15% | 회원이 체감하는 결과 |
| 유연성 | 10% | 부상 예방·자세 보조 |

**총합**: 100%

### 가중치 선정 원칙

1. **위험 관리 최우선**: 부상 안정성이 가장 높은 가중치 (20%)
2. **건강 지표 중심**: 심폐 지구력 + 근지구력 = 40% (거의 고정값)
3. **일반인 중심**: 최대 근력보다 반복 수행 능력 중시
4. **결과 체감**: 체성분 밸런스 15% (회원이 실제 체감하는 지표)

### 목표별 가중치 조정 가이드

**체중 감량 목표**:
- 체성분 밸런스: 20% ↑
- 심폐 지구력: 25% ↑
- 하체 근력: 10% ↓

**근력 향상 목표**:
- 하체 근력: 25% ↑
- 근지구력: 25% ↑
- 체성분 밸런스: 10% ↓

**부상 회복 목표**:
- 부상 안정성: 30% ↑
- 유연성: 20% ↑
- 하체 근력: 10% ↓

## 평가 종목 선정 기준

### 공통 이유 5가지

1. **위험 낮음**: 안전한 평가 방법 우선
2. **시간 짧음**: 20~30분 내 완료 가능
3. **장비 최소**: 기본 장비만으로 평가 가능
4. **회원 설명 쉬움**: 직관적이고 이해하기 쉬움
5. **다음 운동 처방으로 바로 연결됨**: 평가 결과가 바로 프로그램 설계에 활용

➡️ 이 조건을 만족 못 하면 아무리 과학적이어도 현장에선 안 씀

## 평가 등급 체계

### 5단계 등급 (추천)

- **S급 (90-100점)**: 상위 5% - 전문 운동인 수준
- **A급 (80-89점)**: 상위 25% - 우수
- **B급 (60-79점)**: 중간 50% - 보통
- **C급 (40-59점)**: 하위 25% - 미흡
- **D급 (0-39점)**: 하위 5% - 개선 필요

### 3단계 등급 (간소화)

- **상 (70-100점)**: 목표 달성 또는 우수
- **중 (40-69점)**: 평균 수준, 개선 필요
- **하 (0-39점)**: 즉시 개선 필요

### 부상 안정성 특별 등급 (O/△/X)

- **O (정상)**: 모든 동작 제한 없음
- **△ (주의)**: 경미한 불편, 일부 동작 주의 필요
- **X (제한)**: 통증 있음, 특정 동작 제한 필요

## 정적 평가와 동적 평가의 역할

### 정적 평가의 역할

**위험 필터링 및 정보 수집**
- 설문 결과: 통증, 병력 확인
- 체성분: 인바디 측정값 저장
- 체형 정렬: 사진 및 메모 저장
- **점수 계산과 무관**: 정적 평가는 정보 수집 목적

### 동적 평가의 역할

**실제 운동 처방의 핵심**
- 레이더 차트의 주 점수
- 가중치 적용된 종합 점수 계산
- 개인 맞춤 프로그램 설계 기준

### 연계 흐름

```
정적 평가 → 위험 필터링 → 정보 수집
                ↓
동적 평가 → 점수 계산 → 레이더 차트 생성
                ↓
운동 프로그램 설계
```

## 환산 메커니즘

> **참고**: 환산 메커니즘은 추후 구현 예정입니다. 현재는 기본 평가 항목만 사용하며, 환산 로직은 추후 개발될 예정입니다.

### 환산 테이블 구조

정기 평가의 상세 지표를 초기 평가 축 점수로 환산하는 중간 레이어

**예시: 하체 근력 환산**
```
① 정기 평가 실측 데이터
   - 스쿼트 5RM = 체중 × 1.2 → 75점
   - 레그프레스 10RM = 체중 × 2.0 → 65점

② 지표별 점수화 (0~100)
   - 스쿼트 5RM → 75점
   - 레그프레스 10RM → 65점

③ 가중 평균 → 상위 축 반영
   하체 근력 점수 = (스쿼트 × 0.6) + (레그프레스 × 0.4) = 71점
   
➡️ 이 점수가 레이더 차트에 반영
```

### 환산 규칙 테이블 (추후 구현 예정)

```sql
CREATE TABLE assessment_conversion_rules (
  id UUID PRIMARY KEY,
  metric_name VARCHAR(255) NOT NULL,  -- "스쿼트 5RM", "플랭크" 등
  category VARCHAR(50) NOT NULL,      -- STRENGTH, CARDIO, etc.
  weight DECIMAL(3,2) NOT NULL,       -- 가중치 (0~1)
  score_formula JSONB,                 -- 점수 계산 공식
  is_initial_item BOOLEAN DEFAULT FALSE, -- 초기 평가 항목 여부
  created_at TIMESTAMP DEFAULT NOW()
);
```

## 초기 평가 vs 정기 평가

### 초기 평가 (Orientation)

**목적**: 공통 언어 정의 (6개 축 기준선 설정)

**평가 항목**:
- 정적 평가: 설문, 체성분, 체형 정렬
- 동적 평가: 초기 평가 항목만
  - 유연성: 좌전굴, 어깨 가동, 고관절 가동
  - 근지구력: 플랭크, 푸쉬업
  - 심폐: 스텝 테스트
  - 안정성: OHSA
  - 하체 근력: 스쿼트 패턴, 체어 스탠드
  - 체성분: 인바디
- **SBD 1RM ❌** (위험)

**결과**:
- AbilitySnapshot 생성 (6개 축 점수)
- 레이더 차트 기준선 설정

> **참고**: 초기 평가의 세부항목/점수환산은 추후 추가 예정입니다.

### 정기 평가 (3개월 후)

**목적**: 변화 추적 + 근력 측정

**평가 항목**:
- 초기 평가 항목 재평가 (레이더 차트 업데이트용)
- **상세 항목 추가** (그래프 차트용)
  - SBD 1RM (조건부)
  - 기타 상세 지표들

**SBD 측정 조건**:
- 안정성 70점 이상
- 유연성 60점 이상
- 3개월 이상 경과
- 초기 평가 완료

**결과**:
- AbilitySnapshot 생성 (초기 항목 + 상세 항목 환산)
- 레이더 차트 업데이트 (초기 vs 현재)
- 그래프 차트 데이터 (상세 항목별 추이)

> **참고**: 정기 평가의 세부항목/점수환산, 환산테이블은 추후 구현 예정입니다.

## 시각화 구조

### 레이더 차트

**데이터 소스**: `AbilitySnapshot`

**표시 방식**:
- 초기 평가: 회색 선 (기준선)
- 최근 평가: 색상 선 (최신 AbilitySnapshot)
- 모든 평가: 투명도로 히스토리 표시 (선택적)
- **점수 표시**: 각 축에 점수(0-100) 숫자로 표시

**특징**:
- 항상 6축 고정
- 초기 vs 최근 비교
- 성장 추이 시각화

**6개 축**:
1. 하체 근력
2. 심폐 지구력
3. 근지구력
4. 유연성
5. 체성분 밸런스
6. 부상 안정성

**회원 설명**: "이건 성적표가 아니라 운동 설계 지도입니다"

### 그래프 차트

**데이터 소스**: `AssessmentItem`

**표시 방식**:
- 시간별 추이 (X축: 날짜, Y축: 측정값 또는 점수)
- 여러 항목 비교 가능

**예시**:
- SBD 1RM 추이 (스쿼트, 벤치프레스, 데드리프트)
- 플랭크 시간 추이
- 좌전굴 거리 추이
- 인바디 체지방률 추이

> **참고**: 그래프 차트는 추후 구현 예정입니다.
