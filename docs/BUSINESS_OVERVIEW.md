# 헬스 데이터 플랫폼 기획서

> **헬스장 운영자를 위한 회원 관리 시스템 기획서**

---

## 1. 프로젝트 소개

### 현재 상황의 한계점

대부분의 헬스장에서는 회원 정보를 엑셀이나 수기로 관리하고 있어 다음과 같은 문제점이 있습니다:

-   회원의 운동 능력 변화를 추적하기 어려움
-   개별 회원의 성과를 객관적으로 평가하기 어려움
-   시간이 지나면서 기록이 누락되거나 정확도가 떨어짐

### 이 시스템이 해결하는 문제

**"헬스 데이터 플랫폼"**은 회원의 신체 능력을 수치화하고 시간에 따른 변화를 시각적으로 추적할 수 있는 시스템입니다.

---

## 2. 핵심 기능

### 2.1 기본 회원 관리

-   **회원 정보**: 이름, 연락처, 가입일 등 기본 정보 관리
-   **회원권 관리**: 회원권 종류, 구매일, 만료일 추적
-   **PT 횟수 관리**: PT 횟수 등록, 차감, 남은 횟수 확인

### 2.2 회원 능력치 측정 및 추적 (핵심 차별화 기능) ⭐

#### 초기 평가 및 정기 평가

-   **초기 평가**: 회원 가입 시 1회, 모든 비교의 기준점(Baseline) 설정
-   **정기 평가**: 주간/월간 주기로 진행, 변화 추적
-   **6가지 영역 평가**: 하체 근력, 심폐 지구력, 근지구력, 유연성, 체성분 밸런스, 부상 안정성

**예시**: 신규 회원 A씨의 초기 평가 59점 → 3개월 후 정기 평가 72점 (+13점 상승)

#### 능력치 시각화

-   **종합 능력치 차트 (헥사곤)**: 6가지 영역의 현재 상태를 한눈에 파악
-   **시간별 변화 추이 (라인 차트)**: 초기 평가부터 현재까지의 변화 그래프
-   **이전 평가 대비 비교**: 지난 주/지난 달 대비 개선 정도 표시 (+15%, +20% 등)
-   **전체 평균 대비 비교**: 다른 회원들의 평균 대비 본인 위치 확인

#### 부상 이력 관리

부상 이력을 관리하고, 부상이 있는 경우 **해당 영역은 운동 수행이 불가능하므로 평가에서 제외**됩니다:

-   부상 부위, 발생 날짜, 심각도, 회복 상태 기록
-   부상이 있는 영역은 평가에서 제외 (예: 무릎 부상 → 하체 근력 평가 제외)
-   부상이 완치되면 해당 영역 평가 재개

**예시**: 회원 B씨가 무릎 부상을 입은 경우

-   무릎 부상 발생 → 하체 근력 평가 제외 (점수 감점이 아닌 평가 불가 상태 표시)
-   부상 완치 → 하체 근력 평가 재개

---

## 3. 운영상의 장점

-   **회원 상태 파악 용이성**: 전체 회원의 평균 능력치를 한눈에 확인, 능력치 급격히 하락한 회원 자동 식별
-   **데이터 기반 프로그램 제안**: 객관적인 평가를 통한 맞춤형 운동 프로그램 제안, 프로그램 효과 검증
-   **회원 만족도 향상**: 명확한 목표 설정, 그래프로 변화 확인하여 운동 동기 부여
-   **운영 효율성 개선**: 체계적인 기록 관리, 시간 절약, 계산 오류 최소화

## 4. 피드백 요청 사항

실제 헬스장 운영 경험을 바탕으로 아래 사항에 대한 피드백을 부탁드립니다.

1. **현재 회원 정보를 어떻게 관리하고 계신가요?** (엑셀, 수기 노트, 기타 프로그램 등)

2. **회원의 운동 능력을 평가하는 시스템이 있나요?** (있다면 방식, 없다면 필요한 방식)

3. **이 시스템과 현재 운영 방식의 가장 큰 차이점은 무엇이라고 생각하시나요?**

4. **추가로 필요한 기능이 있다면 무엇인가요?** (예: 출석 관리, 결제 관리 등)

5. **다음 기능들 중 개발 우선순위를 매겨주세요.**

    - [ ] 회원 정보 관리 (기본)
    - [ ] 회원권 및 PT 횟수 관리
    - [ ] 능력치 평가 시스템
    - [ ] 능력치 시각화 (그래프, 차트)
    - [ ] 부상 이력 관리
    - [ ] 대시보드 (전체 현황)

6. **능력치 평가 주기는 어떻게 생각하시나요?** (회원 가입 시, 주간, 월간, 분기별 등)

7. **이 시스템을 사용할 때 가장 우려되는 점은 무엇인가요?**

8. **기타 건의사항이나 의견이 있으시면 자유롭게 적어주세요.**
